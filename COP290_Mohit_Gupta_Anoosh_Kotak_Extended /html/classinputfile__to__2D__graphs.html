Member Function Documentation

graph2D inputfile_to_2D_graphs::generate_2D_graph ( )

The input file will consist of 3 lists and each element of a list will consists of 2 co-ordinates having edges between them.

The co-ordinates of the first list will be stored in the Vertex set of the first graph. The edges will be stored in the edge set after giving them weight according to the distance(which will be calculated) between 2 vertices. Similarly, the graphs for other lists will also be generated. The graphs generated will be objects graph2D class.
