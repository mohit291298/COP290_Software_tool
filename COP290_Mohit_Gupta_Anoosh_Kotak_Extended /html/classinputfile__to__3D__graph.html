Member Function Documentation

graph2D inputfile_to_3D_graph::generate_3D_graph ( )

The input file will consist of a list and each element of the list will consists of 2 co-ordinates having edges between them.

The co-ordinates will be stored in the Vertex set of the graph. The edges will be stored in the edge set after giving them weight according to the distance(which will be calculated) between 2 vertices. The graph generated will be an object of graph3D class.
