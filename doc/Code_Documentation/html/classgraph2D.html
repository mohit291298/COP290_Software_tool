void graph2D::filter_hidden_lines ( graph3D  g,
plane  p 
)

A Member function filters hidden lines from the 2D graph.

This function will always be called after calling the make_2D_graph_with_vertices and join_2D_edges function. Here, first we convert the 2D coordinates of 2D graph into 3D vertex, then we see if the projected 3Dvertex and 3Dvertex are on same side of any face in face set. If yes, then edges from the vertex are done hidden.

Parameters
gThe 3D graph which contains the set of faces.
pThe projected plane
